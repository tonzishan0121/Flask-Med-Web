# 🖥️ Vue医疗预测数据看板 
**Vue-Med-Dashboard** 📊👨⚕️  

✨ 基于Vue3+ElementPlus构建的医疗预测可视化看板，对接Flask预测服务，学校云计算课程前端项目 ✨

---

## 🌟 功能特性
- 📱 响应式布局适配多端
- 📈 可视化预测结果数据（柱状图/雷达图）
- 📋 交互式病例数据输入表单
- 🔗 Axios异步请求管理
- 🎨 主题色定制与切换支持
- 📤 预测结果PDF导出功能

---

## 🛠️ 服务对接
- 心脏病预测数据可视化
- 糖尿病预测多维图表
- 肺癌预测结果对比分析
- 实时模型准确率看板
